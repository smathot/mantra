Mantra connect

This plug-in establishes a connection to Mantra and opens a new Mantra logfile.

Options

  • Tracker attached [Yes | No] By setting this option to 'No', you can test your experiment even if a Mantra server is not available.

  • Logfile [string] The name of the Mantra logfile. Mantra may prepend underscores to this name, if a file with the desired name already exists.

  • Host [string] A hostname or IP-address of the server on which Mantra is running.

  • Port [integer] The port number at which Mantra is listening. You can find this number in the top-right of the Mantra preview window.

More information

You can find more information about Mantra on http://www.cogsci.nl/mantra.
